기술 범례

※ 본 공략에서 <휴먼/요마/몬스터> 기술 데이터와 그 외의 기술 데이터의 서술 형식이 다릅니다.

성능

  • 공격 속성 : 방패 회피와 속성 방어를 참조할 수 있는 값
  • 명중 타입(상) / 기본 명중율(하) : 타입에 따라서 명중의 조건이나 명중 계산식이 다르다. 명중율은 기본적인 명중 확률이 된다. [] 안의 수치는 명중 확률 상승량에 영향을 준다.
  • 기본 위력 : 스테이터스에 영향을 받지 않는 데미지의 기본값
  • 성장도 : 스테이터스에 영향으로 상승하는 데미지량을 결정
  • 효과 : 해당 기술을 사용해서 발생하는 효과
  • 추가 기술 : 아래에 설명


추가 기술
추가 기술을 가지는 기술은 이하 세 가지 타입이 있다.

  1. 주 기술로 메인 타겟을 공격하고, 추가 기술로 그 주변을 공격하는 타입
  2. 주 기술과 추가 기술이 같이 메인 타겟을 공격하는 타입
  3. 주 기술과 추가 기술이 같이 메인 타겟을 공격하고, 재차 추가 기술로 그 주변을 공격하는 타입
메인 타겟에 발동하는 추가 기술은 카운터나 방패에 의한 회피 판정을 받지 않는다. (내성 무시는 불가능, 명중 판정도 통상과 동일)
추가 기술에 따라서 동명의 기술과 속성이나 효과가 조금 다른 것이 있다.

상세

  • 속도 : <퍼스트 트릭>은 행동 순서가 빨라진다. <딜레이 액션>은 행동 순서가 느려진다.
  • 방패 사용 : 해당 기술을 사용한 턴에, 자신이 방패 회피를 발동시킬 수 있을지 여부
  • 특성 : 해당 기술의 특성. 대응하는 내성을 가진 상대에게 무효화된다.
  • 마법 방패 : 음술 '섀도우 서번트'의 회피 대상
  • 회피 : <디플렉트, 십자 막기, 스웨이 백>의 회피 대상
  • 카운터 : <안개청안, 상신무상, 급소 던지기, 드래곤 스크류, 다연장 시스템, 환마상파, ○○ 배리어, 카운터 피어, 요술 '유리의 방패'>의 카운터 대상
  • 사격 카운터 : '반응 사격'의 카운터 대상
  • 방패 : 방패의 회피 대상. ()는 회피 확률 감소를 나타낸다.
  • 특수 효과 : 해당 기술의 특수한 효과
  • 사용 조건 : 해당 기술을 사용하기 위한 조건
